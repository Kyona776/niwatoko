install_requires=[
    "numpy",
    "nltk",
    "tensorflow",
    "torch",
    "transformers",
    "pillow",
    "streamlit",
    "gradio",
]

# - `nltk`: 自然言語処理ライブラリ
# - `tensorflow`: 機械学習ライブラリ
# - `torch`: 機械学習ライブラリ
# - `transformers`: 自然言語処理の事前学習済みモデルを提供するライブラリ
# - `pillow`: 画像処理ライブラリ
# - `streamlit`: Webアプリケーション開発ライブラリ
# - `gradio`: Webアプリケーション開発ライブラリ
# 
# これらのライブラリは、要件定義書に記載された機能を実現するために必要となります。パッケージのインストール時に、これらの外部ライブラリも一緒にインストールされるようにしています。
